# Website-Soff
Châm ngôn của Website Save once, find forever.
Đây là website về lưu trữ những thước film, manga, manwa, animen donghua, comic,.... Khi bạn lưu trữ sẽ giúp bạn tốn ích thời gian sau này tìm và khi muốn xem lại.
Mọi thứ đều dễ dàng khi bỏ ra 1 ích thời gian để lưu lại những thế giới điện ảnh mà mình yêu thích.
